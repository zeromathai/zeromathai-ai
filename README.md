# ZeromathAI — Complete AI Study Roadmap

**ZeromathAI** is a structured AI learning roadmap designed like a textbook.

It connects artificial intelligence, machine learning, deep learning, language AI, vision AI, multimodal AI, reinforcement learning, optimization, and AI hardware into one continuous learning path.

This repository is not a code library.  
It is a **curriculum map** for understanding how AI concepts connect.

## 📚 Recommended Learning Path

Follow this order if you're starting from scratch:

1. AI Overview  
2. Search Algorithms  
3. Neural Networks  
4. Machine Learning  
5. Deep Learning  

Then expand into:

- Language AI  
- Vision AI  
- Multimodal AI  
- Reinforcement Learning  
- Optimization  

This roadmap is designed to move from **foundations → systems → applications**.
