## 🧩 Core Structure

### 1. Why RNN is Needed

- [Recurrent Neural Network (RNN)](https://zeromathai.com/en/recurrent-neural-network-en/)

👉 Standard neural networks:

- process inputs independently  

👉 RNN:

- preserves previous information  
- processes sequences  

---

### 2. Core Mechanism (Hidden State)

- [RNN Architecture](https://zeromathai.com/en/recurrent-neural-network-en/)

👉 Uses:

- current input  
- previous hidden state  

👉 This enables memory across time steps.

---

### 3. Deep Architecture

- [Deep RNN](https://zeromathai.com/en/deep-rnn-en/)

👉 Extends:

- single-layer RNN → multi-layer RNN  

👉 Enables more complex sequence modeling.

---

### 4. Position in Deep Learning

- [Recurrent Neural Network](https://zeromathai.com/en/recurrent-neural-network-en/)  
- [CNN](https://zeromathai.com/en/convolutional-neural-networks-en/)

👉 RNN:
- sequential / temporal data  

👉 CNN:
- spatial / image data
