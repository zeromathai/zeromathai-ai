## 🧩 Core Structure

### 1. Game Search Foundation

- [State Space Search](https://zeromathai.com/en/state-space-search-en/)

👉 Defines:

- possible states  
- actions  
- transitions  

👉 Game problems extend this with opponent interaction.

---

### 2. Minimax (Core Principle)

- [Minimax Algorithm](https://zeromathai.com/en/minimax-algorithm-en/)

👉 Key idea:

- maximize your outcome  
- minimize opponent advantage  

👉 Assumes optimal play from both sides.

---

### 3. Heuristic Evaluation

- [Heuristic Function](https://zeromathai.com/en/heuristic-function-en/)

👉 Used when:

- full-depth search is impossible  
- intermediate states must be evaluated  

---

### 4. Alpha-Beta Optimization

- [Alpha-Beta Pruning](https://zeromathai.com/en/alpha-beta-pruning-en/)

👉 Improves efficiency by:

- pruning unnecessary branches  
- preserving optimal decision  

---

### 5. Expansion to General Search

- [A* Algorithm](https://zeromathai.com/en/a-algorithm-en/)  
- [Informed Search](https://zeromathai.com/en/informed-search-system-en/)

👉 Connects game search to:

- general problem-solving  
- heuristic-based search
