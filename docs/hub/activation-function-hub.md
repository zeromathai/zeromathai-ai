## 🧩 Core Structure

### 1. Why Activation Functions Are Needed

- [Activation Function](https://zeromathai.com/en/activation-function-en/)

Without activation functions, neural networks:

- remain linear  
- cannot increase expressive power  

👉 Activation functions introduce **nonlinearity**, enabling deep learning.

---

### 2. Why Linearity Is Not Enough

- [Linear Activation Function](https://zeromathai.com/en/linear-activation-function-en/)

Even deep networks collapse into:

- a single linear transformation  

👉 Nonlinearity is the key to model expressiveness.

---

### 3. Sigmoid vs ReLU

- [Sigmoid](https://zeromathai.com/en/sigmoid-en/)  
- [ReLU](https://zeromathai.com/en/rectified-linear-unit-en/)

👉 Sigmoid:
- outputs between 0 and 1  
- interpretable as probability  

👉 ReLU:
- simple and efficient  
- dominant in modern deep learning  

---

### 4. Output Layer Design

- [Softmax Function](https://zeromathai.com/en/softmax-function-en/)

👉 Converts outputs into:

- probability distributions  
- multi-class predictions
