## 🧩 Core Structure

### 1. State Space Search (Starting Point)

- [State Space Search](https://zeromathai.com/en/state-space-search-en/)

Search is based on:

- states  
- actions  
- transitions  
- goals  

👉 This defines the full problem-solving structure.

---

### 2. DFS vs BFS (Core Strategies)

- [DFS](https://zeromathai.com/en/dfs-en/)  
- [BFS](https://zeromathai.com/en/bfs-en/)

👉 DFS:
- explores deeply along one path  

👉 BFS:
- explores level by level  

---

### 3. Hybrid Strategy

- [IDS](https://zeromathai.com/en/ids-en/)

👉 Combines:

- DFS memory efficiency  
- BFS completeness  

---

### 4. Informed Search (Efficiency Upgrade)

- [Informed Search](https://zeromathai.com/en/informed-search-system-en/)  
- [Heuristic Function](https://zeromathai.com/en/heuristic-function-en/)

👉 Uses additional information to:

- guide search  
- improve efficiency  

---

### 5. Greedy Search (Intuitive Heuristic)

- [Greedy Search](https://zeromathai.com/en/greedy-search-en/)

👉 Selects the most promising node:

- based only on heuristic  

---

### 6. A* Algorithm (Practical Solution)

- [A* Algorithm](https://zeromathai.com/en/a-algorithm-en/)

👉 Combines:

- path cost (g)  
- heuristic estimate (h)  

👉 Finds optimal paths efficiently.
